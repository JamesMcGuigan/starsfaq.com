Stuart Douglas' Stars! Utilities

  • Pirates Retreat [Home Page for these utilities]

  • Xtreem Borders 1.6.5 - Includes VB6 libraries [2.6 Mb]
      Mapper program - allows players to combine and analyse data extracted from the game, including the visualization of each players influence over a territory, mapping of stargate networks and the display of allies scanner data, while providing a convenient place for keeping notes on the other races in the game (Note: for best results in the stars.ini file, under [misc] add newreports=1)

  • Xtreem Borders 1.6.8 Update [91 kb] (Requires VB6 libraries)
      Update for the above program, including a fix for a bug when combining data from allies

  • Xtreem StarMail 1.3.0 [32 kb] (Requires VB6 libraries)
      A utility to help hosts manage the sending of turns to players (both MAPI and SMPT supported), allows for optional attachments, custom messages and alternate email addresses

  • StarEd 1.2 - Stars Editor [53 kb] (Requires VB6 libraries)
      Complete component and hull editor for Stars! with a an easy to use interface (works with both I and JRC3 patches), modifications can be saved into a small .MOD file, which can be distributed to players to modify their own exe's with. (Note: It's best to keep an unedited copy of stars on your harddisk)

  • Generic Mod Files for StarsEd
    • Unedited MOD File [31 kb] - Data from an unmodified .exe (Requires StarsEd )

  • Beta Mod Files for StarsEd
    • Viper STM File - v0.2.3 beta - by Coyote [31 kb] (Requires StarsEd )
    • Vulture MOD File - by Adam Oddy [31 kb] (Requires StarsEd )
    • VML MOD File - v0.9b - by Verker [16 kb] (Requires StarsEd )

    Note: The above programs require that you have the Visual Basic 6 libraries installed, if you don't have them they are part of the XTreem Borders 1.6.5 package

Other Stars! Utilities

  • Packet Calculator [20 kb]
      Program for calculating the impact and effects of packets at different speeds and at different sizes.

  • Battlesim Testbed [58 kb]
      Set of game files for conducting battlesim testbeds in, allows for most PRT/LRT combos and all races are at max tech with all MT components. Unzip into a new directory and edit the links to suit your own directory structure.

  • Universe Creator 1.3 [273 kb]
      Utility for creating custom universe setups from a bitmap image and also allows for the repositioning of homeworlds
  • Art Lathrop's Attractivness Calculator Spreadsheet [7 kb] (Excel File - compatible with Quattro Pro)
      Spreadsheet for calculating the relitive attractivness (for targeting order during battle) of ship designs
  • SB Possey's Stars Spreadsheet [108 kb] (Excel File - partially compatible with Quattro Pro)
      Large spreadsheet for calculating most of the guts within stars, including jamming, movement, cloaking, colony projection, planet habitability, tech trade probabilities, research timescales, minefields and more
  • Karl Peninger's Stars! Terraforming Spreadsheet [530 kb] (Excel File - contains Visual Basic Scripts)
      Spreadsheet for calculating the most optimal order for terraforming planets in an alliance.
